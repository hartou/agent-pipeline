I just published `@hartou/agent-pipeline` to npm: a config-driven multi-agent delivery runner that lets Copilot act as the client/QA layer, Fugu coordinate the work, and implementation workers like DeepSeek-4-pro and gpt-4o-mini build in isolated Docker containers and git worktrees. The latest release adds containerized orchestration, safer brownfield upgrades with `--upgrade`, branch/worktree isolation for worker changes, Fugu validation before client approval, and a refreshed runtime architecture diagram. Install it with `npx @hartou/agent-pipeline init --target . --skill`, then configure your repo, add provider keys locally, and run `node tools/agent-runner/run.mjs doctor` to validate the setup.
