I published `@hartou/agent-pipeline`, a multi-agent npm package for structuring AI coding work across a client agent, an orchestrator, and implementation workers. The architecture diagram shows the core runtime flow: Copilot shapes the request and handles QA, Fugu coordinates the plan and validates worker output, and worker agents run in isolated containers/worktrees before anything is accepted. The package is meant to make multi-agent development easier to install, reason about, and reuse across real repositories. Install with `npx @hartou/agent-pipeline init --target . --skill`.
