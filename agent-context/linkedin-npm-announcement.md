I just published `@hartou/agent-pipeline` to npm. It is a small toolkit for turning AI coding into a more structured team workflow: one agent helps clarify the request, another coordinates the work, and specialist agents implement changes in a controlled way before anything is accepted. The goal is simple: make multi-agent development easier to install, easier to review, and safer to use in real repositories, including existing projects. You can try it with `npx @hartou/agent-pipeline init --target . --skill` and I’ll add the architecture diagram in the post.
